## Project Overview

This Weather App was built by me using HTML, CSS, and JavaScript, through my knowledge and understanding from AltSchool live classes, and online resources. It allows users to search for any city in the world and view current weather conditions as well as a 5-day weather forecast. The application uses the Open-Meteo Geocoding API to retrieve city coordinates and the Open-Meteo Forecast API to fetch weather information.

## Features

* Search weather by city
* Current weather display
* 5-day forecast
* Weather icons and descriptions
* Error handling
* Responsive design
* Search history
* Geolocation support

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Open-Meteo API
